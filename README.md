# 🎓 Student Performance Prediction System

## Overview
This project predicts student academic performance using Machine Learning and Streamlit.

## Features
- 📊 Predict student marks
- 📈 Attendance analysis
- 🤖 AI recommendations
- 📉 Interactive charts
- 🎯 Performance classification

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-Learn
- Plotly
- Joblib

## Dataset Features
- Study Hours
- Attendance
- Assignments Completed
- Previous Marks

## How to Run

```bash
pip install streamlit pandas scikit-learn plotly joblib
streamlit run app.py
